## Commands list  

**❗ Argunments in `[ ]` belowed are optional**  

| Command name   | Usage               | Example              | Description                        |
|----------------|---------------------|----------------------|------------------------------------|
|`.setwowsregion`| `.setwowsregion <region>` |`.setwowsregion na` | `region`: `asia`, `na`, `eu`, `ru`<br>The default region is **ASIA** if its empty in commands, <br>it can change the default value in this server, <br>***only admin can use it***|
|`.wowsregion`   | `.wowsregion` | `.wowsregion` | Check the default region value in this server
| `.link`      | `.link <region> <ign>`   | `.link asia B2U` | Link your wows and discord account, after linked, <br>all the `[region] [ign]` below can be replaced by `me` |
| `.wws`       | `.wws [region] <ign>`     | `.wws asia B2U`<br>`.wws B2U`<br>`.wws me`  | Show the player's stats overview |  
| `.wws ship`       | `.wws [region] <ign> <ship>`     | `.wws asia B2U Ise`<br>`.wws B2U Slava`<br>`.wws me Yamato`  |Show the player's specific warship stats |
| `.recent`       | `.recent [region] <ign> [battleType] [days]`     | `.recent asia B2U`<br>`.wws B2U 7`<br>`.wws me rank 21`  | Show the player's last few days stats, <br>`[days]`: `1`~`21`, its default by 1 if empty, <br>`[battleType]`: `rank`, `solo`, `div2`, `div3`, its default by `pvp` if empty |
| `.invite` |||The link for invite this bot|
